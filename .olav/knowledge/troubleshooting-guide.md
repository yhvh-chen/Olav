# 常见故障和解决方案

## BGP 邻居状态异常

**症状**: BGP 邻居显示 Idle / Active

**可能原因**:
1. 网络连接中断
2. 防火墙阻止 TCP 179 端口
3. 路由器配置错误
4. 邻居 IP 地址不可达

**解决步骤**:
1. Ping 邻居 IP 地址
2. 检查防火墙规则
3. 验证 BGP 配置
4. 查看 BGP 日志

## 接口无法正常通信

**症状**: 接口物理层 UP，但 IP 不可达

**可能原因**:
1. IP 地址配置错误
2. 子网掩码不匹配
3. VLAN 配置问题
4. VLAN Trunk 配置错误
